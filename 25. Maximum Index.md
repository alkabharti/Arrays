### Solution :

Time Complexity :


```java
static int maxIndexDiff(int A[], int N) 
{ 
    int i,j,max=0,current=0;
    for(i=0;i<N;i++)
    {
        j=N-1;
        while(j>i)
        {
            if(A[i]<=A[j])
            {
                current=j-i;
                if(current>max)
                    max=current;
                break;
            }
            j--;
        }
    }
    return max;
}

```
