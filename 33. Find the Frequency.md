### Solution :

- Time Complexity : 


```java
int findFrequency(int A[], int x)
{
    ArrayList<Integer> arr = new ArrayList<Integer>();
    int i,count=0;
    for(i=0;i<A.length;i++)
    {
        arr.add(A[i]);
        if(arr.contains(x))
        {
            count++;
            arr.remove(Integer.valueOf(x));
        }
    }
    return count;
}

```
