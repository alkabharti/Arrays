Solution :

Time Complexity :


```java
int rowWithMax1s(int arr[][], int n, int m) 
{
    int i,j;
    int max=0,index=0;
    int count=0;

    for(i=0;i<n;i++)
    {
        count=0;
        for(j=0;j<m;j++)
        {
            if(arr[i][j]==1)
                count++;
        }
        if(count>max)
        {
            max=count;
            index=i;
        }
    }
    if(max==0)
        return -1;
    return index;
}

```
