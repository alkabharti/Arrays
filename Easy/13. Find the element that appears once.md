## Find the element that appears once

### Solution 1: 

- Time Complexity : O(N)
- Space Complexity : O(1)


```java
public static int search(int A[], int N)
{
    int ans=0;
    for(int i=0; i<N; i++) {
        ans=ans^A[i];
    }
    return ans;
}

```


### Solution 2: 

- Time Complexity : O(logN)
- Space Complexity : O(1)


```java
public static int search(int A[], int N)
{
    int low = 0;
    int high = N-1;
    while(low <= high) {
        int mid = (low+high)/2;
        if(mid%2 == 1 && A[mid] == A[mid-1] || mid%2 == 0 && A[mid] == A[mid+1]) 
        {
            low = mid+1;
        }
        else
            high = mid-1;
    }
    return A[low];
}

```
