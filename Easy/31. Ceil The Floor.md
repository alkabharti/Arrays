## Ceil The Floor

### Solution : 

- Time Complexity : O(N)


```java
Pair getFloorAndCeil(int[] arr, int n, int x) {
    int ceil = Integer.MAX_VALUE;
    int floor = 0;
    for(int i=0; i<n; i++) {
        if (arr[i] == x)
        {
            floor = arr[i];
            ceil = arr[i];
        }
        if (arr[i] <= x && arr[i] > floor)
            floor = arr[i];
        else if (arr[i] >= x && arr[i] < ceil)
            ceil = arr[i];
    }
    if (ceil == Integer.MAX_VALUE)
        ceil = -1;
    if (floor == 0)
        floor = -1;

    return new Pair(floor,ceil);
}

```
