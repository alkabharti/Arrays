### Solution : Sliding Window

- Time Complexity : O(N)
- Space Complexity : O(1)


```java
int findZeroes(int arr[], int n, int m) {
    int i=0, j=0;
    int count=0, maxlen=0;
    while(j<n){
        if(count<=m){
            if(arr[j]==0) 
                count++;
            maxlen=Math.max(maxlen, j-i);
            j++;
        }
        else{
            if(arr[i]==0) 
                count--;
            i++;
        }
    }
    if(count<=m) 
        maxlen=Math.max(maxlen, j-i);
    return maxlen;
}

```
