### Solution :

- Time Complexity :


```java
int findMin(int arr[], int n)
{
    int low = 0;
    int high = n-1;
    while(low < high)
    {
        int mid = (low + high) / 2;
        if (arr[mid] > arr[high])
            low = mid + 1;

        else if(arr[mid] < arr[high])
            high = mid;

        else
            high--;
    }
    return arr[high];
}

```
