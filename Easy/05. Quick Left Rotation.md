## Quick Left Rotation

### Solution : 

- Time Complexity : O(N)
- Space Complexity : O(1)


```java
void leftRotate(long arr[], int k,int n)
{
    k = k%n;
    reverse(arr, 0, k-1);
    reverse(arr, k, n-1);
    reverse(arr, 0, n-1);
}
void reverse(long[] arr, int start, int end)
{
    int i = start, j = end;
    while(i<j){
        long temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
        i++;
        j--;
    }
}

```
