### Solution : 

- Time Complexity : 
- Space Complexity : 


```java
void leftRotate(long arr[], int k,int n)
{
    while(k < n) {
        long first_ele = arr[0];
        for(int i=0; i<n-1; i++) {
            arr[i] = arr[i+1];
        }
        arr[n-1] = first_ele;
    }
}

```
