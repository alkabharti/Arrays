### Solution :

- Time Complexity :


```java
int minSubset(int[] Arr,int N) 
{ 
    long sum=0;
    for(int i=0;i<N;i++)
    {
        sum+=Arr[i];
    }
    Arrays.sort(Arr);
    long t=0;
    int ans=0;
    for(int i=N-1;i>=0;i--)
    {
        t+=Arr[i];
        sum-=Arr[i];
        ans+=1;
        if(sum<t) 
            return ans; 
    }
    return ans;
}

```
