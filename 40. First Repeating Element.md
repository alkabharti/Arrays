### Solution :

Time Complexity :


```java
public static int firstRepeated(int[] arr, int n) 
{
    int i,j;
    for(i=0;i<n;i++)
    {
        j = n-1;
        int element = arr[i];
        while(i<j)
        {
            if(element == arr[j])
            {
                return i+1;
            }
            else
                j--;
        }
    }
    return -1;
}

```
