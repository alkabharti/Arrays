<h3> Solution : </h3>

Time Complexity : O(NlogN)

```java
String printLargest(String[] arr) 
{
    Arrays.sort(arr, new Comparator<String>(){
        public int compare(String a, String b)
        {
            String n1=a+b;
            String n2=b+a;
            return n2.compareTo(n1);
        }
    });
    String str="";
    int i;
    for(i=0;i<arr.length;i++)
        str+=arr[i];
    return str;
}
```
