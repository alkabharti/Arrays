### Solution 1 :

Time Complexity : O(n)


```java
long maxSubarraySum(int arr[], int n)
{
    int i;
    long current=arr[0],max=arr[0];
    for(i=1;i<n;i++)
    {
        current=Math.max(current+arr[i],arr[i]);
        max=Math.max(max,current);
    }
    return max;
}
```

### Solution 2 :

Time Complexity : O(n)


```java
long maxSubarraySum(int arr[], int n)
{
    int i;
    long max=Integer.MIN_VALUE, current=0;
    for(i=0;i<n;i++)
    {
        current+=arr[i];
        if(current>max)
            max=current;
        if(current<0)
            current=0;
    }
    return max;
}
```
