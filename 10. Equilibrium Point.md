<h3>Solution : </h3>

Time Complexity : O(n)

```java  
public static int equilibriumPoint(long arr[], int n) 
{
    int i,left=0,right=0,res=0;
    for(i=0;i<n;i++)
        right+=arr[i];
    for(i=0;i<n;i++)
    {
        right-=arr[i];
        if(left==right)
            res=i+1;
        left+=arr[i];
    }
    if(res==0)
        return -1;
    else
        return res;
}
```
