Solution :

Time Complexity :


```java
long countStrings(int n) 
{
    // Fibonacci sequence
    int f=1;
    int s=2;
    int t=0;

    if(n==0)
        return f;
    if(n==1)
        return s;

    for(int i=2;i<=n;i++)
    {
        t=(f+s) % 1000000007;
        f=s;
        s=t;
    }

    return t;
}
```
